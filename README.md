# antiVirusCheck
用来检测目标beacon是否存在杀软，cs本地判断，无须联网，无须上传任何文件
源码中%sr字典可自行添加收集到的杀软进程和名称
## 关于乱码
如果出现返回结果乱码，用编辑器打开的时候选择保存在本地的编码格式为utf-8
